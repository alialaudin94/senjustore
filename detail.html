Rincian Joki Satuan

Joki BR (Battle Royale) Per Match

  • Bronze - Silver: FREE / per match
  • Silver - Gold: Rp 1.000 / per match
  • Gold - Platinum: Rp 1.500 / per match
  • Platinum - Diamond: Rp 2.000 / per match
  • Diamond - Master: Mulai Rp 3.000 / per match

*Match gratis hanya berlaku untuk 5 match pertama di rank Bronze-Silver.

Joki CS-R (Clash Squad Ranked) Per Bintang

  • Bronze - Silver: FREE / per bintang
  • Silver - Gold: Rp 1.000 / per bintang
  • Gold - Platinum: Rp 1.500 / per bintang
  • Platinum - Diamond: Rp 2.000 / per bintang
  • Diamond - Master: Mulai Rp 3.000 / per bintang

*Bintang gratis hanya berlaku untuk 5 bintang pertama di rank Bronze-Silver.

  • Platinum - Heroic: Rp 35.000
  • Diamond - Heroic: Rp 30.000
  • Heroic - Master Kuning: Rp 50.000
  • Master Kuning - Grandmaster: Rp 55.000

Paket Joki BR (Battle Royale) ke Grandmaster

  • Platinum - Heroic: Rp 35.000
  • Diamond - Heroic: Rp 25.000
  • Heroic - Master Kuning: Rp 50.000
  • Master Kuning - Grandmaster: Rp 55.000

Bagaimana Jika Rank Saya Tidak Ada di Daftar?

Jika Anda ingin order dari rank lain (misalnya Silver ke Grandmaster, atau Gold ke Master), silakan ikuti langkah berikut:

  1. Hubungi Admin via **WhatsApp (+62 857-0468-1654)**.
  2. Sebutkan Rank Awal dan Rank Tujuan Anda (contoh: BR, Silver I ke Grandmaster).
  3. Kami akan membuatkan **Paket Harga Custom** yang dihitung berdasarkan gabungan harga per match/per bintang dari rank awal hingga tujuan Anda.
  4. Pembayaran dapat dilakukan setelah kesepakatan harga custom.
